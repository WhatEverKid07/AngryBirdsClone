# (Original Repo) Angry Birds style game

![Angry Birds style game](https://i.ibb.co/fQBKr9M/K-pkiv-g-s.png)

An effort to replicate some of the levels and relevant mechanisms of Rovio's famous Angry Birds game, built in Unity game engine. Source code is provided for educational purposes.

### Assets

Graphics: all sprites and graphic elements are upscaled versions of the original pc clients files and some of the newest phone version

# (New Section) Level Maker

This is a custom level maker that I created, with assets from the base game as the foundation I added a new option to create your own levels and be able to place blocks, birds, and pigs and be able to save your level and be able to share with other people for them to play your level.

## Basic Controls

- Click & Drag to place/move any object.
- Right click any object to delete it.
- Scroll wheel on an object to rotate it.

## Future

I hope to be able to add many more blocks, birds and pigs, along with the addition of themes, and possible small quality of life features.
